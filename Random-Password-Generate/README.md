# 🔐 Random Password Generator

A simple and responsive **Random Password Generator** built using **HTML, CSS, and JavaScript**.  
This application allows users to generate secure passwords with customizable options and view password strength in real time.

---

## 🚀 Features

- Generate secure random passwords
- Adjustable password length using a slider
- Option to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Real-time **password strength meter**
- Copy password to clipboard with one click
- Clean and responsive user interface

---

## 🧠 How It Works

1. Select the **password length** using the slider.
2. Choose the character types you want:
   - Uppercase
   - Lowercase
   - Numbers
   - Symbols
3. Click **Generate Password**.
4. The application creates a random password based on the selected options.
5. The **strength meter** analyzes the password using:
   - Password length
   - Character variety
6. Click the **copy icon** to copy the password to the clipboard.

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Password generation logic and interactivity

---


## 🔑 Password Strength Logic

The password strength meter evaluates the password using:

- Password **length**
- Presence of **uppercase letters**
- Presence of **lowercase letters**
- Presence of **numbers**
- Presence of **symbols**

Based on the score, the password is classified as:

- 🔴 **Weak**
- 🟡 **Medium**
- 🟢 **Strong**
