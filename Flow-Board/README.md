# FlowBoard

FlowBoard is a **Kanban-style task management board** built using **HTML, CSS, and JavaScript**.  
Users can create tasks, drag them between workflow columns, and the data is saved using **LocalStorage**, so tasks remain after page refresh.

## Features
- Add new tasks with title and description
- Drag and drop tasks between columns
- Delete tasks
- Task counter for each column
- Data persistence using LocalStorage

## Workflow Columns
- To Do
- In Progress
- Done

## Tech Used
- HTML
- CSS (Flexbox)
- JavaScript
- Drag & Drop API
- LocalStorage
