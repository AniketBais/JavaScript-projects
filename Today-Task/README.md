# Today-Task

Today-Task is a simple productivity project that helps users organize and manage their daily tasks. The goal of this project is to provide an easy way to keep track of tasks that need to be completed today.

## Features
- Add daily tasks
- Mark tasks as completed
- Simple and clean interface
- Helps improve daily productivity

## Purpose
This project was created to practice development skills while building a useful tool for managing everyday tasks.

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Add a task you want to complete.
2. View the list of tasks for the day.
3. Mark tasks as completed after finishing them.
