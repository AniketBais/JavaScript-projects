# Snake Game 

A classic Snake Game built using HTML, CSS, and JavaScript.

## Features
- Snake movement using arrow keys
- Score tracking
- High score saved in LocalStorage
- Apple food
- Game over modal

## How to Play
Use Arrow keys to move the snake and eat the apple.
